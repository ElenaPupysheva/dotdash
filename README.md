# DotDash

**Точки и тире** — Android-приложение для изучения азбуки Морзе.  
Проект разрабатывается на **Jetpack Compose** с архитектурой **MVI** и разделением на слои по принципам **Clean Architecture**.

## О проекте

Приложение помогает изучать азбуку Морзе в удобном интерактивном формате.  
Пользователь может просматривать символы в справочнике, проходить тренировки, отслеживать прогресс и настраивать параметры обучения.

Проект создаётся как учебный pet-project с упором на:
- современный Android-стек,
- реактивный UI на Compose,
- предсказуемое управление состоянием через MVI,
- чистую архитектуру,
- масштабируемую структуру проекта.

## Основные возможности

### Первая версия
- главный экран с разделами приложения;
- справочник символов азбуки Морзе;
- поиск по символам;
- фильтрация по категориям;
- экран деталей символа;
- тренировочный режим;
- экран результатов тренировки;
- статистика обучения;
- настройки приложения;
- сохранение пользовательских настроек локально.

### Планируемые улучшения
- несколько режимов тренировки;
- избранные и сложные символы;
- история сессий;
- звуковое воспроизведение кода Морзе;
- вибрация;
- аудиодиктант;
- обучение словам и фразам;
- ежедневные задания.
